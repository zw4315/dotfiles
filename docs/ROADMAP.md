# Roadmap

## Current Focus

- Keep docs minimal and non-duplicated.
- Stabilize Neovim keymap/workflow around Diffview + Snacks.
- Keep module install flow reproducible on Ubuntu.

## Next

- Improve cross-platform profile split (Linux / WSL / macOS).
- Standardize private/local overlay handling.
- Add a lightweight verification script for core links and key commands.

## Later

- Evaluate Vim/Neovim feature parity boundary.
- Introduce optional capability profiles (minimal/default/full).
- Reduce long-tail legacy shell aliases into machine-local overrides.
